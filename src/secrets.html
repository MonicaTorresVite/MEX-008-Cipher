Secrets...

¿Sabes guardar un secreto?

Secrets es el mejor sitio web para mantener tus secretos, ya sabes, secretos... Ya que somos el único sitio web que encripta tus secretos y en el cual poder descifrarlos depende completamente de ti. No olvides leer las instrucciones antes de usar la aplicacion, para que puedas aprovechar al máximo lo que Secrets tiene para ti.

Secrets no se hace responsable por el mal uso que se le pueda dar a la aplicacion.

COMO ENCRIPTAR SECRETOS: Simplemente ingresa en la caja de texto indicada tu secreto o el texto que desees encriptar en letras mayúsculas, despues elige un numero para que sea la clave con la que encriptaremos tu secreto, y oprime el boton que dice "ENCRIPTAR SECRETO", posteriormente, tu secreto sera mostrado en una caja de texto inferior ya encriptado de una manera segura para que simplemente copies el mensaje y se lo puedas mandar a quien va dirigido.

MUY IMPORTANTE: No olvides decirle al destinatario el numero clave con el cual encriptaste tu secreto, ya que este es el único modo con el cual podra revelarlo.

COMO REVELAR SECRETOS: Para revelar tu secreto, únicamente pega el texto que desees descifrar en la caja de texto indicada y escribe el numero clave con el que fue encriptado, posteriormente optime el boton que dice "REVELAR SECRETO" y una vez hecho esto el secreto se revelara en la caja de texto inferior.
